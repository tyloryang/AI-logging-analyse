> **按角色和专业水平组织的精选技能集合。** 不知道从哪里开始？从下面选择一个包，获取适合您角色的精选技能集。

## 🚀 快速开始

1. **安装仓库：**

   ```bash
   npx antigravity-awesome-skills
   # 或手动克隆
   git clone https://github.com/sickn33/antigravity-awesome-skills.git .agent/skills
   ```

2. **从下面列表中**根据您的角色或兴趣**选择您的包**。

3. **在AI助手中引用来使用技能：**
   - Claude Code: `>> /skill-name 帮我...`
   - Cursor: `@skill-name 在聊天中`
   - Gemini CLI: `Use skill-name...`
   - Codex CLI: `Use skill-name...`

## 🔧 使用建议

### 🎯 如何选择包

1. **新手入门：** 从"基础"包开始
2. **按角色选择：** 选择最符合您工作的包
3. **按项目需求：** 根据当前项目选择相关包
4. **逐步扩展：** 掌握基础后探索其他包

### 🔄 包组合策略

**全栈开发者：**
```
基础包 + React向导包 + API构建者包 + QA工程师包
```

**AI开发者：**
```
基础包 + AI开发者包 + 数据库专家包 + Docker专家包
```

**安全工程师：**
```
基础包 + Web安全专家包 + 渗透测试员包 + 后端开发包
```

### 💡 最佳实践

1. **不要贪多：** 一次专注1-2个包
2. **实践为主：** 边学边用
3. **循序渐进：** 从简单到复杂
4. **定期复习：** 巩固已学技能
