Codex × 微信接入控制台

这个页面负责完成二维码登录、凭据落盘、以及 Docker 容器内 Codex + CLI-WeChat-Bridge 的启动与停止。聊天仍然发生在微信里，不在这个页面里渲染。

这是一个管理页，不是公网多租户产品。上线前请至少放到受保护的内网、VPN 或反向代理鉴权之后。

1. 当前状态

显示微信凭据状态、当前工作目录，以及桥接服务是否已经在容器内运行。

2. 微信扫码登录

点击“生成二维码”后，用微信扫码。页面会自动轮询状态，扫码确认成功后自动写入容器内凭据文件。

等待扫码…

3. 启动 Codex 接入

填写容器内工作目录，然后点击启动。后台会先启动 bridge，再拉起 companion，并将微信消息转入 Codex。

启动成功后，去微信里直接给桥接机器人发消息即可。

4. 实时日志

显示 bridge、companion、以及容器内 Codex 的合并日志，便于排查二维码、凭据或启动问题。
